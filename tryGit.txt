就是随便写点东西集美们体验一下用 git 多人协作的快乐
内容合并这个只有纯文本才可以，Word 文档是二进制格式的用不了，只能整个文件上传
大概意思就是可以几个人同时编辑一个文档，然后他可以自动把每个人修改的部分合并到一起
如果两个人修改的内容有冲突，那么在合并之前需要手动处理冲突
一般出了什么问题百度一下都比较容易解决
总之只要不在不清楚的情况下用各种 -f 之类的强制选项，一般就不会出现进度丢失之类的惨剧

在本地 add 和 commit 之后 push 到云端
如果这个时候别人先 push 过其他内容的话会报提醒，需要先 pull 一下，之后再 push
如果别人修改了同一个文件，合并出现了冲突，则需要手动解决
